# Pokemon Generator

## In a Word
Generating new pokemon images with DCGAN.

## Source of Data
Pokemon Wikipedia(https://wiki.52poke.com).

## Model
DCGAN[[1]](#1) built in PyTorch(https://pytorch.org/).
